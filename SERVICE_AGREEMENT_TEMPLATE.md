## 2. 项目概述 / Project Overview

### 2.1 项目描述
甲方将为乙方开发一个**豪华车修理行专业网站**，该网站为纯前端静态网站，采用现代化设计和响应式布局，支持英文和中文双语切换。

**This project involves developing a professional luxury auto repair website as a static front-end application with modern design, responsive layout, and bilingual support (English/Chinese).**

### 2.2 技术栈
- 前端框架：React 18
- 路由系统：React Router 6
- 构建工具：Vite
- 样式系统：CSS Modules + 设计令牌系统
- 托管平台：Vercel 或 Netlify（甲方选择最优方案）
